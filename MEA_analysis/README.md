# RBCM-edge MEA Analysis

This directory contains the reproducible MEA analysis code for the RBCM-edge paper.

## Formal Stimulus Names

Some raw files and early exploratory scripts used `single edge` and `double edge`.
The manuscript uses the following formal names:

- `single edge` -> `UME`: Uniform-background moving edge
- `double edge` -> `CME`: Contextual-background moving edge

All formal analysis scripts, tables, reports, and figures should use `UME` and `CME`.

## Formal Main Analysis

The current formal MEA result is generated by:

```powershell
python D:\study\project\RBCM-Edge\MEA_analysis\run_MEA_final_UME_CME_trajectory_analysis.py
```

Fixed analysis settings:

- Coordinate transform: none; original unit coordinates are used.
- Spatial region: full retinal recording area.
- Paired retina:
  - `pair_31_32`: CME `000031`, UME `000032`
  - `pair_34_35`: CME `000034`, UME `000035`
  - `pair_37_38`: CME `000037`, UME `000038`
- Motion directions: R, RU, U, LU, L, LD, D, RD.
- Movement window: approach-to-center.
- UME steps: 0-6 out of 13.
- CME steps: 0-5 out of 11.
- Response metric: `log((ON + 0.10) / (OFF + 0.10))`.
- Grid scales: 8x8, 12x12, 16x16.
- Valid grid rule: at least 3 UME good units and at least 3 CME good units.
- Main statistical test: within-grid label permutation test for grid-level trajectory MAE.
- Multiple-comparison correction: Benjamini-Hochberg FDR across the 72 formal conditions.

## Interpretation Boundary

This is a spatially matched grid-level local population analysis.

It compares local RGC sorted-unit population response trajectories between UME
and CME recordings from the same retinal preparation. It is not a unit-level
paired comparison, because sorted units cannot be reliably matched one-to-one
across separate recordings and spike-sorting results.

Recommended manuscript wording:

> Spatially matched grid-level analysis of paired retinal recordings showed that
> UME and CME stimulation evoked distinct local RGC sorted-unit population
> response trajectories across multiple directions and grid scales.

Avoid wording such as:

- the same RGC changed its response between UME and CME;
- each grid was statistically significant;
- CME globally enhances or suppresses RGC responses.
